# Simple Loop Patterns

## Project Overview

This beginner-friendly Python project helps students understand how
nested `for` loops work by generating different star (`*`) patterns. It
strengthens logical thinking and provides a foundation for solving
pattern-based programming problems.

------------------------------------------------------------------------

## Objective

Practice the following programming concepts:

-   `for` Loop
-   Nested `for` Loop
-   `range()` Function
-   `print()` Function
-   `end=""` Parameter
-   Pattern Printing Logic

------------------------------------------------------------------------

## Technologies

-   Python 3
-   Jupyter Notebook / Google Colab

------------------------------------------------------------------------

## Features

-   Print an increasing star pattern.
-   Print a decreasing star pattern.
-   Learn row and column iteration using nested loops.
-   Understand how loop variables control pattern generation.

------------------------------------------------------------------------

## How to Run

1.  Open the notebook in Jupyter Notebook or Google Colab.
2.  Run all cells.
3.  Observe the generated star patterns.

------------------------------------------------------------------------

## Concepts Covered

-   `for` Loop
-   Nested `for` Loop
-   `range()` Function
-   `print()` Function
-   `end=""` Parameter
-   Pattern Printing Logic

------------------------------------------------------------------------

## Learning Outcomes

After completing this project, students will be able to:

-   Use nested `for` loops effectively.
-   Generate different text-based patterns.
-   Understand row and column iteration.
-   Improve logical thinking and problem-solving skills.
-   Build a strong foundation for advanced loop-based programming.
